# How to Build
1. Get npm
2. cd to directory
3. Run ``npm install``
4. Run ``grunt`` and keep this running as you develop (To manually build, run ``grunt build``)
7. The app is now running in dev mode and you can open http://localhost:3000 in your browser
5. You are ready to dev!
